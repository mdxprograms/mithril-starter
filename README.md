# Mithril Starter
![mithriljs](https://avatars2.githubusercontent.com/u/19475707?s=280&v=4)


### Up and running
`nvm use`

`npm i`

### Dev
`npm start`

### Build
`npm run build`

### App structure

```bash
src
├── components
│   └── Nav.js
├── index.html
├── index.js
├── models
│   └── User.js
├── router.js
└── views
    ├── HomeView.js
    ├── Layout.js
    └── UserView.js
```

```markdown
* components - (examples: Nav, Footer, UserList)
* models - (Model/State => examples: User, Article)
* views - (Main pages => examples: HomeView, UsersView)
```
